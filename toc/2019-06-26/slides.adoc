=== Adding help with docstrings

[source,python]
----
def dir2(o):
    "Return a list of attributes not inherited from the `object` class"

    return sorted(list(set(dir(o)) - set(dir(object))))
----

[source,python]
----
>>> help(dir2)
Help on function dir2 in module __main__:

dir2(o)
    Return a list of attributes not inherited from the `object` class
----
